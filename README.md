# praveen-pravi6B.dev

A sleek, responsive portfolio website built with Vue.js 3 and Vite, featuring a modern design with glassmorphism effects and smooth animations.

-   **Modern Aesthetic**: A clean, professional design focusing on typography and visual hierarchy.
-   **Glassmorphism UI**: Uses `GlassCard` components for a modern, frosted-glass look.
-   **Interactive Sections**:
    -   **Home**: Introduction and welcome screen with dynamic interactions.
    -   **About**: Personal background and information.
    -   **Skills**: specific technical skills and tools.
    -   **Projects**: Showcase of development projects.
    -   **Contact**: Information for reaching out.
-   **Responsive Design**: optimized for various screen sizes, from mobile to desktop.
-   **Social Integration**: Seamless links to professional profiles via `SocialLinks`.

## üì¶ Project Setup

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd vue-portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The app will run at `http://localhost:5173` (by default).

4.  **Build for production:**
    ```bash
    npm run build
    ```
